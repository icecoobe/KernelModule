# TestProcEntry 

Test `read` and `write` in the `file_operations`. Note that APIs related Proc system change from 3.10, see detail in the reference.

## Module Compilation

```sh
sudo make
```

## Module Cleanup

```sh
sudo make clean
```

## Module Installation

```sh
sudo make install
```

## Module Uninstallation

```sh
sudo make uninstall
```
